# Sahil Al Nojoom · Dispatch

Transport dispatch system for Sahil Al Nojoom (Saudi Arabia).

- **Admin Panel** — `/admin/` — create orders, issue access keys, manage drivers/companies, quotations, reports.
- **Driver App** — `/driver/` — drivers accept trips and update status. Login by access key.
- **Company App** — `/company/` — companies confirm orders and track status. Login by access key.

Real-time backend on Firebase (Cloud Firestore). Currency: SAR.
